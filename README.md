# python-rustserver
Rust Server Automation Only!


# Install Requirements
Install ansible! 
* Amazon Linux
    * `sudo amazon-linux-extras install ansible2`
* Python3
    * `sudo yum install python3.x86_64`

### Python modules
`pip install -r requirements.txt`

`pip3 install -r requirements.txt`

# Commands
* `-h, --help`
    * show this help message and exit
* `--install`
    * Install Rust Server
* `--run`
    * Run Rust Server
* `--stop`             
    * Stop Rust Server
* `--restart`
    * Restart Rust Server
* `--update`
    * Update Rust Server
* `--install_mod`
    * Install Oxide
* `--command`
    * Run Rcon Command
* `--partial_wipe`
    * Clean up Maps
* `--full_wipe`
    * Clean up everything!
* `--clean`
    * Wipes out Rust Server Completely
